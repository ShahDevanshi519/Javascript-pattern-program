# JavaScript Pattern Programs

This repository contains various pattern programs implemented using JavaScript. These programs helped me practice nested loops, conditions, and pattern logic.

## Patterns Covered

### 1. Left Triangle Star Pattern

```text
*
**
***
****
*****
```


### 2. Reverse Left Triangle Star Pattern

```text
*****
****
***
**
*
```


### 3. Number Triangle Pattern

```text
1
12
123
1234
12345
```

### 4. Continuous Number Triangle Pattern

```text
1
23
456
78910
1112131415
```

### 5. Repeated Number Triangle Pattern

```text
1
22
333
4444
55555
```

### 6. Reverse Number Triangle Pattern

```text
12345
1234
123
12
1
```

### 7. Mixed Character Patterns

```text
*
*#
*#*
*#*#
*#*#*
```

### 8. Alphabet Triangle Pattern

```text
A
AB
ABC
ABCD
ABCDE
```

### 9. Repeated Alphabet Triangle Pattern

```text
A
BB
CCC
DDDD
EEEEE
```

### 10. Continuous Alphabet Triangle Pattern

```text
A
BC
DEF
GHIJ
KLMNO
```

### 11. Reverse Alphabet Triangle Pattern

```text
ABCDE
ABCD
ABC
AB
A
```
